# Computers
Basics about Computers

- What is it? Information processor that takes an input and provides an output
  - Memory can be used to store the information
- Computer Program: Programmed set of processes

## Stack
- Applications
- [Operating System](./Operating_System/Operating_System.md): Provides basic input, output, storage & processing
- BIOS: Link between the operating system software & hardware
  - Also known as *firmware*
- Hardware: 
  - Motherboard
  - Central Processing Unit (CPU)
  - Graphics Processing Unit (GPU)
  - Random Access Memory (RAM)
  - Storage: Solid State Drive (SSD) or Hard Disk Drive (HDD)
