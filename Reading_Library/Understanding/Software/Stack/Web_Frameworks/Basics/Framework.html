Relative Units

Parent Default Font Size: 16px


    Relative to Root/Parent Element

  • em: Size of parent (2em: 32 px)
  • ex: x-height of the element's font (2ex: 14.3594 px)
  • ch: Width of the character 0 (1ch: 9px)
  • rem: Font Size of the root element (1rem: 32px)
  • lh: Line height of the element: Does Not Work

    • Relative to Viewport (Responsive)

  • vw: 1% of the viewport's width (2vw: Variable/Responsive)
  • vh: 1% of the viewport's height (2vh: Variable/Responsive)
  • vmin: 1% of the viewport's smaller dimension (2vmin: Variable/Responsive)
  • vmin: 1% of the viewport's larger dimension (2vmax: Variable/Responsive)
