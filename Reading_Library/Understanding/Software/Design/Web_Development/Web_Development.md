# Web Development

### Specifications & Standards
Specifications are just what certain browsers will classify as valid.  There is no set standards group for html, css and web based DOM javascript.

- [DOM](https://html.spec.whatwg.org/)
- [Html](https://html.spec.whatwg.org/)
- [CSSOSM](https://www.w3.org/TR/cssom-1/#introduction)
- [Javascript (ES6)](https://262.ecma-international.org/6.0/)
- [Worldwide Web Consortium (W3)](https://www.w3.org/Consortium/)
