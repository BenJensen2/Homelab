# Bash Scripting
Writing scripts within the Bash shell

### Location of commands
- /bin (Base Commands) & /usr/bin (Extras added on)

### Cool commands
- date
- cal
- pwd
- ls

### Desired Commands
- hl: Home Lab Access
    - o: Open in VS Code
    - t: List file index/tree
- ds: cd to Dashboard directory
